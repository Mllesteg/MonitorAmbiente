Monitoreo de Temperatura, Humedad y Hidrogéno

Objetivos:

  • Recolectar datos de temperatura, humedad y hidrogéno en tiempo real utilizando el microcontrolador ESP32 y el sensor DHT11.
  • Transmitir estos datos a la plataforma ThingSpeak para visualización.
